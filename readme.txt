Herramienta de Dither 

Herramienta de degradado (dithering) de imágenes sencilla y personalizable, enfocada en la claridad y la facilidad de uso.

Descripción general
Dither Tool es una aplicación ligera basada en el navegador para convertir imágenes utilizando diferentes técnicas de dither y paletas de colores personalizadas.

Está diseñada para ser directa, flexible y fácil de modificar.

Características
Múltiples algoritmos de dither:

Floyd–Steinberg

Atkinson

Jarvis–Judice–Ninke

Stucki

Dither ordenado:

Bayer 4×4

Bayer 8×8

Soporte para paletas de colores personalizadas

Paletas retro integradas (Game Boy, NES, C64, etc.)

Generación automática de paletas (K-Means)

Resolución ajustable para un acabado estilo pixel-art

Control de intensidad para el efecto de dither

Exportación del resultado en PNG

Importación/exportación de paletas en JSON

Uso
Abre la herramienta en tu navegador.

Sube una imagen.

Elige o crea una paleta de colores.

Selecciona un algoritmo de dither y sus ajustes.

Haz clic en "Procesar Imagen".

Descarga el resultado.

Ejecución del proyecto
No requiere instalación.

Simplemente abre el archivo index.html en cualquier navegador web moderno.

El proyecto se ejecuta totalmente en el lado del cliente y es fácil de modificar. Puedes:

Añadir nuevos algoritmos de dither.

Modificar las paletas existentes.

Ajustar los estilos de la interfaz (UI).

Extender la funcionalidad según sea necesario.

Notas
El procesamiento se realiza localmente en el navegador.

El rendimiento puede variar dependiendo del tamaño de la imagen.
